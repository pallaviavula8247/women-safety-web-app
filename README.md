# Women Safety Web App

A responsive Women Safety Web App built using HTML, CSS, and JavaScript.

## Features

* SOS Alert Button
* Emergency Contact Management
* Safety Awareness Interface
* Responsive Design
* Modern User Interface

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

Women_Safety_Web_App/

├── index.html

├── style.css

├── script.js

└── README.md

## How to Run

1. Download or clone the repository.
2. Open the project folder in VS Code.
3. Open `index.html` using Live Server.
